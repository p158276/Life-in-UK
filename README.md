# Life in the UK Quick Study Guide

An interactive bilingual (Chinese/English) study guide website for the **Life in the UK** citizenship test.

## Features

- **18 Topic Sections** covering history, politics, law, culture, religion, sports, landmarks, and more
- **75 Interactive Flashcards** with flip animation, mastery tracking, and shuffle
- **46 Wrong Answer Markers** highlighting commonly mistaken exam topics with detailed explanations
- **Chinese/English Bilingual** one-click language switching with localStorage persistence
- **Category Filtering** browse by History, Politics & Law, Culture & Arts, Society & Life
- **Search** find any topic instantly across all sections
- **Wrong Answers Mode** dedicated filter to focus on previously incorrect questions
- **Responsive Design** works on desktop, tablet, and mobile

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion (animations)
- shadcn/ui components
- Vite (build tool)

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Design

British Heritage Editorial style with:
- Navy blue (#1B2A4A) + warm gold (#C8A96E) + cream (#FAF8F5) colour palette
- Playfair Display (headings) + Source Sans 3 (body) typography
- Magazine-style layout with sidebar navigation
